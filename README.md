# BRIM Marketing Campaign Tracker API

A simple Django REST API to manage clients and their marketing campaigns with authentication, CRUD, and reporting.

## Features
- JWT Authentication
- Client management
- Campaign tracking
- Client-specific reports

## Stack
- Django REST Framework
- SQLite (for development)
- JWT Authentication

---

## Setup Instructions
```bash
git clone https://github.com/Mr-iyke/BRIM-Marketing-Campaign-Tracker-API.git
cd brim-marketing-api
pip install -r requirements.txt
python manage.py runserver
